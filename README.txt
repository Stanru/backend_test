Создание jar файла:
Установить Gradle
Для восстановления wrapper в корне проекта выполнить команду gradle wrapper
В корне проекта выполните команду gradlew jar
файл будет создан в директории ./build/libs

Запуск:
Выполните команду java -jar ./build/libs/backend_test-1.jar

Использование:
В соответствии с заданием.
Ввод данных в формате "user -> email.@email.com", построчно.
Для вывода результата нужно ввести пустую строку, вместе с этим программа завершит выполнение.
